# Student Planner

Мобильное приложение для организации учебного процесса и тайм-менеджмента студента: расписание занятий с учётом замен, список задач, дедлайны и заметки в одном месте.

> Дипломный проект, разработанный в **Екатеринбургском колледже транспортного строительства (ЕКТС)**.

---

## О проекте

**Student Planner** — Android-приложение, созданное в качестве дипломного проекта. Цель — помочь студентам структурировать учебный процесс: вести расписание, отслеживать задания и сроки их сдачи, работать с данными даже без подключения к интернету.

## Возможности

- **Расписание занятий** — просмотр пар по дням недели с интеллектуальным слиянием базового расписания и замен.
- **Управление задачами** — создание, редактирование и отметка дел, жестовое управление (свайпы).
- **Дедлайны** — контроль сроков сдачи работ.
- **Оффлайн-режим** — полноценная работа без интернета с локальным хранением данных.
- **Авторизация и регистрация** — личный аккаунт с выбором группы при регистрации.

## Технологии

### Мобильный клиент
- **Язык:** Kotlin
- **UI:** Jetpack Compose (декларативный UI)
- **Архитектура:** MVVM (Model-View-ViewModel)
- **Асинхронность:** Kotlin Coroutines + Flow
- **Внедрение зависимостей:** Hilt
- **Сеть:** Retrofit 2
- **Локальная БД:** Room Persistence Library
- **Среда разработки:** Android Studio

### Серверная часть
- **Бэкенд:** ASP.NET Core (C#) — REST API, фоновый парсинг Excel/PDF с изменениями расписания

### Проектирование
- **Макеты интерфейса:** Figma

## Установка

### Из готового релиза (APK)

1. Перейди в раздел [Releases]({{ https://github.com/xckrt/studentplanner/releases/tag/Release }}).
2. Скачай файл `studentplanner.apk`.
3. На устройстве разреши установку из неизвестных источников.
4. Открой скачанный APK и установи приложение.

> При установке Android может показать предупреждение о том, что пакет не зарегистрирован верифицированным разработчиком — это связано с новыми требованиями Google и не влияет на работу приложения для учебного использования.

### Сборка из исходного кода

```bash
git clone {{ ссылка на репозиторий }}
```

1. Открой проект в Android Studio.
2. Дождись синхронизации Gradle.
3. Запусти сборку: **Build → Build APK(s)** или **Run** на устройстве/эмуляторе.
